# **App Name**: QuantumVerse

## Core Features:

- Modular Learning Tracks: Progressive quantum computing curricula with interactive modules covering superposition, entanglement, and complex algorithms.
- Intelligent Quantum Professor: An AI tutor that serves as a tool to answer complex questions, provide personalized study hints, and simplify quantum jargon based on user mastery levels.
- Live Circuit Lab: A drag-and-drop visual interface to build and simulate quantum circuits, providing real-time state change visualizations.
- Quantum Playground IDE: In-browser coding editor supporting Qiskit and Cirq snippets for hands-on programming challenges.
- Dynamic XP & Rank System: Gamification engine that tracks streaks and levels users through ranks like Qubit Explorer and Entanglement Master.
- Generative Study Pathways: AI-powered tool that analyzes progress to dynamically generate personalized quizzes and curriculum adjustments.
- Persistent Learning Storage: Secure storage of user progress, code snippets, and rank history in a Cloud SQL database.

## Style Guidelines:

- Primary color: Electric Violet (#9933FF) to evoke the energy of quantum excitations and high-tech elegance.
- Background color: Obsidian Indigo (#0C0A14), a deep, desaturated dark shade that provides depth for glow effects.
- Accent color: Celestial Cyan (#33D6FF) used for highlighting data states and interactive UI triggers.
- Headline font: 'Space Grotesk' for its technical and scientific feel. Body font: 'Inter' for clarity in long-form educational text.
- Linear neon icons with subtle glassmorphic backgrounds and 'inner glow' effects to maintain a futuristic aesthetic.
- Modular bento-style dashboard with frosted glass (glassmorphism) containers and high vertical contrast.
- Parallax particle effects representing quantum background radiation and smooth Framer Motion transitions between modules.
